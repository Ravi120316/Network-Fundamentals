# Network Fundamentals

A comprehensive collection of notes covering the fundamental concepts of computer networking.  
This repository is designed for students, learners, and enthusiasts to understand networking basics.

## Topics Covered:
- What is a Network and its Importance
- Types of Networks: LAN, MAN, WAN, PAN
- Network Topologies: Star, Ring, Bus, Mesh
- Network Devices: Router, Switch, Hub, NIC
- Basic Protocols and Communication
- Key Concepts: Bandwidth, Latency, Throughput, Error Handling
- Overview of OSI and TCP/IP models

## Objective:
To provide a structured and easy-to-understand reference for learning computer networking from scratch.
